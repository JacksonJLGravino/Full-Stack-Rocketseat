# LP de Clube de Assinatura - Full-Stack - Rocketseat

O projeto é um site desktop de clube de assinatura de livros com informações sobre o produto.
Esse é um desafio prático da formação Full-Stack, um dos conteúdos disponíveis para alunos da Rocketseat.

## 🛠 Tecnologias

O projeto foi construído com:

- HTML
- CSS

## ✨ Projeto

O foco desse projeto são as animações, usando o transition, animation e keyframes, para criar animações personalizadas para o projeto
